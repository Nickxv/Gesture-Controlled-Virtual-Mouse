# AI Gesture Mouse Controller – Basic User Guide

## Running the Program

1. Install required libraries: pip install opencv-python mediapipe pyautogui PyQt5

2. Run the program: python main.py

3. Click **Start Camera** to begin using gesture controls.

# Gesture Controls

 Gesture                  | Action      
 ------------------------ | ----------- 
 Index Finger             | Move Cursor 
 Thumb + Index            | Left Click  
 Thumb + Little Finger    | Right Click 
 Open Hand + Move Up/Down | Scroll      
 Closed Fist              | Pause       
 Thumbs Up                | Screenshot  

Screenshots are saved automatically in the program folder.

# Adjusting Cursor Speed
Use the **Cursor Speed slider** to increase or decrease mouse movement speed.

# Stopping the Camera
Click **Stop Camera** to turn off gesture detection.

# Tips
   Keep your hand clearly visible to the camera.
   Use good lighting for better detection.
